iCal Combinator

Congratulations! You stumbled across the application for combining iCalendar calendars that we carefully try to keep secret as much as possible.

However, since you are now are here, we might as well tell you what it is all about.

What is it all about?

iCalendar is marching along quite nicely. The trouble is, you easily end up having to track numerous .ical feeds and hook them all up with - say - Google Calendar. That's a lot of labour. Time you better spend on something a little more interesting.

How does it work?

Pretty simple. iCal Combinator allows you to integrate calendars from the commandline. It offers a simple restful API for merging calendars at will.

The Calendars

In iCal Combinator, every calendar is identified by a URL like this:

http://ical-combinator.appspot.com/calendars/{key}

There is no access control, so you if you create your own calendar, you better choose your key wisely.

Defining a Calendar

To create a calendar, simply PUT a list of URLs to iCalendar feeds to the calendar location of your preference. That's it.

Obtaining the definition of your Calendar

You can always obtain of a Calendar by performing a GET on the Calendar's URL.

Obtaining the iCalendar feed

... but the interesting stuff is of course to obtain an integrated iCalendar feed. There are two ways. Neither one of them is particularly complicated, but let's start with the simple one: simply do a GET on the location of your Calendar, and append '/ical'. So the full URL would be something like this:

http://ical-combinator.appspot.com/calendars/{key}/ical

This way works pretty well, especially if you want to hook your iCalendar feed up with some existing tools.

However, if you are a real restafarian, then you consider the approach above one for sissies. Instead, you want the real thing: Content Negotiation! Yeah, baby! Now, iCal Combinator supports that as well. If you simply do a GET on the calendars URL and pass an 'Accept: text/calendar' header, you will get your calendar as an integrated calendar feed.
